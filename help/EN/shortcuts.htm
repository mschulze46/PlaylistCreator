Keyboard shortcuts
 
The following table presents the keyboard shortcuts of Playlist Creator 3.6.2:
Add folder Ctrl + O
Add files Ctrl + F
Add playlist Ctrl + P
Select all items Ctrl + A
Deselect all items Ctrl + D
Move selected items up Ctrl + Up
Move selected items down Ctrl + Down
Move selected items to the top Ctrl + Home
Move selected items to the end Ctrl + End
Remove selected items Ctrl + Del
Remove all items Ctrl + Shift + Del
Refresh item status F5
