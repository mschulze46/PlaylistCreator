  1. Erstellung einer Playlist

    Fügen Sie der Playlist zuerst alle gewünschten Lieder hinzu. Wählen Sie anschließend den Speicherort der Playlist aus. Nun muss nur noch der Name der Playlist eingegeben und der gewünschte Playlisttyp ausgewählt werden. Durch Klick auf "Playlist erstellen" kann die Playlist nun erstellt werden.


  2. Einträge zu einer Playlist hinzufügen

    Es gibt verschiedene Möglichkeiten Einträge zu einer Playlist hinzuzufügen: Klicken Sie auf die Schaltfläche "Ordner hinzufügen", um den Inhalt eines gesamten Ordners hinzuzufügen oder auf die Schaltfläche "Dateien hinzufügen", um ein oder mehrere Dateien zur Playlist hinzuzufügen. Möchten Sie den Inhalt einer bereits existierenden Playlist hinzufügen, klicken Sie auf die Schaltfläche "Playlist hinzufügen" und wählen Sie die gewünschte Playlist aus.

    Ordner, Dateien und Playlists können auch per Drag&Drop vom Windows Explorer hinzugefügt werden. Bitte beachten Sie, dass Playlists nur einzeln hinzugefügt werden können (Single Selection)!

    Die ausgewählten Dateien werden der Playlist nur dann hinzugefügt, wenn sich der betreffende Dateityp in der Liste der aktiven Dateitypen befindet (Einstellungen >> Dateitypen). Ein kleines Beispiel: Der Ordner den Sie hinzufügen möchten enthalte MP3- und OGG-Dateien. In der Liste der aktiven Dateitypen sei der Dateityp 'MP3' enthalten, der Typ 'OGG' jedoch nicht. Fügen Sie der Playlist diesen Ordner hinzu, so werden alle MP3-Dateien in die Playlist aufgenommen, alle OGG-Dateien werden nicht aufgenommen.


  3. Existierende Playlist-Dateien öffnen oder einfügen

    Es gibt zwei Methoden, um den Inhalt einer existierenden Playlist zu importieren: Die Playlist öffnen oder einfügen.

    Eine Playlist öffnen bedeutet, dass die momentane Playlist verworfen und der Inhalt der zu öffnenden Playlist eingefügt wird. Außerdem wird der Speicherort, der Name und der Typ der Playlist automatisch mit der zu öffnenden Playlist initialisiert. Diese Methode bietet sich also an, wenn eine existierende Playlist editiert werden soll.

    Es gibt zwei Varianten, um eine Playlist zu öffnen: Klicken Sie entweder in der Toolbar auf die Schaltfläche "Öffnen" und wählen Sie die gewünschte Playlist aus, oder werfen Sie eine einzelne Playlist per Drag&Drop vom Windows Explorer auf die Playlist und klicken Sie anschließend auf die Schaltfläche "Öffnen".

    Eine Playlist einfügen bedeutet, dass der Inhalt der Playlist an der ausgewählten Stelle der momentanen Playlist eingefügt wird. Die momentane Playlist wird also nicht verworfen, sondern um den Inhalt der eingefügten Playlist ergänzt. Auch der Speicherort, der Name und der Typ der momentanen Playlist werden nicht verändert.

    Auch beim Einfügen von Playlisten gibt es zwei Varianten: Klicken Sie entweder auf die Schaltfläche "Playlist hinzufügen" und wählen Sie die gewünschte Playlist aus, oder werfen Sie eine einzelne Playlist per Drag&Drop vom Windows Explorer auf die Playlist und klicken Sie anschließend auf die Schaltfläche "Einfügen".


  4. Ordner rekursiv zu einer Playlist hinzufügen

    Ordner können sowohl mittels Drag&Drop vom Windows Explorer, als auch mittels der Schaltfläche "Ordner hinzufügen" rekursiv (d.h. Inhalt des Ordners und aller Unterordner) zur Playlist hinzugefügt werden.

    Um Ordner mittels Drag&Drop rekursiv hinzuzufügen, lassen Sie den entsprechenden Ordner mit gedrückter "Strg"-Taste auf die Playlist fallen. Natürlich können auch mehrere Ordner und/oder Dateien gleichzeitig auf die Playlist fallengelassen und rekursiv hinzugefügt werden.

    Möchten Sie einen Order mittels der Schaltfläche "Ordner hinzufügen" rekursiv zur Playlist hinzufügen, so müssen Sie lediglich im Ordner-Auswahldialog die Checkbox "Ordner rekursiv hinzufügen", welche sich direkt unter der Anzeige des momentan ausgewählten Ordners befindet, auswählen und mit 'Ok' bestätigen.


  5. Reihenfolge der Playlisteinträge ändern

    Wählen Sie die Einträge aus, die Sie verschieben möchten und klicken Sie auf die Schaltfläche "Nach oben verschieben" bzw. "Nach unten verschieben". Die Einträge der Playlist können zudem auch per Tastenkürzel verschoben werden: Mittels der Tastenkombination "Strg + Pfeil oben" werden die ausgewählten Einträge nach oben bzw. mit "Strg + Pfeil unten" nach unten verschoben. Dieselbe Funktionalität bietet auch das Kontextmenü der Playlist. Ein Rechtsklick auf den markierten Einträgen läßt das Kontextmenü erscheinen und man kann die gewünschte Aktion auswählen.

    Außerdem ist es möglich, die markierten Einträge an den Anfang bzw. an das Ende der Liste zu verschieben. Dies funktioniert analog wie im obigen Abschnitt beschrieben per entsprechender Schaltfläche, Tastenkombination oder Kontextmenüeintrag.

    Die Auswahl mehrerer Einträge erfolgt genau wie im Windows Explorer: Einfach die entsprechenden Einträge mit gedrückter "Shift"- bzw. "Strg"-Taste anklicken.


  6. Einträge von der Playlist entfernen

    Klicken Sie entweder auf die Schaltfläche "Markierte Einträge entfernen" um die momentan ausgewählten Einträge von der Playlist zu entfernen, oder auf die Schaltfläche "Alle Einträge entfernen" um alle Einträge der Playlist zu entfernen. Die markierten Einträge können zudem auch mit dem Tastenkürzel "Strg + Entf" entfernt werden. Um alle Einträge zu entfernen verwenden Sie das Tastenkürzel "Strg + Shift + Entf". Eine weitere Möglichkeit, Einträge von der Playlist zu entfernen, bietet das Kontextmenü der Playlist. Ein Rechtsklick in der Playlist bzw. auf den markierten Einträgen läßt das Kontextmenü erscheinen und man kann die gewünschte Aktion auswählen.

    Bei all diesen Operationen werden lediglich die betreffenden Einträge aus der Playlist entfernt. Die zugehörigen Dateien auf der Festplatte werden selbstverständlich nicht gelöscht!

    Die Auswahl mehrerer Einträge erfolgt genau wie im Windows Explorer: Einfach die entsprechenden Einträge mit gedrückter "Shift"- bzw. "Strg"-Taste anklicken.


  7. Dynamische Titel erzeugen

    Um die Erzeugung dynamischer Titel zu aktivieren klicken Sie in den Einstellungen auf den Unterpunkt "Erweiterte Informationen". Aktivieren Sie dort die Optionen "Erweiterte Informationen schreiben" und "Daten von Tags auslesen". Direkt darunter kann nun das gewünschte Titelformat eingegeben werden. Die zur Verfügung stehenden Titelvariablen werden auf der rechten Seite des Dialogs aufgelistet.

    Hinweis: Dynamische Titel können nur bei Dateitypen erzeugt werden, welche die Einbettung von Tags (Metadaten) unterstützen. Dazu gehören beispielsweise die Dateitypen MP3, OGG und WMA.


  8. Eigene Dateitypen hinzufügen

    Klicken Sie in den Einstellungen auf den Unterpunkt "Dateitypen". Geben Sie dort auf der rechten Seite den neuen Dateityp ein und klicken Sie auf die Schaltfläche "Neu". Der von Ihnen eingegebene Dateityp wird nun der Liste der inaktiven Dateitypen hinzugefügt. Wollen Sie diesen Dateityp sofort aktivieren, so wählen Sie den Typ aus und klicken Sie auf die Schaltfläche "Hinzufügen".


  9. Wo finde ich die grünen Hüte?

    Direkt neben den goldenen Fallschirmen!
