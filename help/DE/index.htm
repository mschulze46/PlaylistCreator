Playlist Creator 3.6.2
 
Der Name ist Programm - Mit Playlist Creator können Playlisten einfach und schnell erstellt werden. Die Zusammenstellung und Erstellung einer Playlist geschieht in sekundenschnelle: Einfach die gewünschten Dateien hinzufügen, einen Speicherort sowie einen Namen für die Playlist eingeben und einen Moment später ist die erste Playlist auch schon fertig! Am schnellsten und einfachsten geht das Hinzufügen der Dateien übrigens mittels Drag&Drop vom Windows Explorer.

Playlist Creator unterstützt die leistungsstarke Open-Source Library MediaInfo. Somit kann ein zuverlässiges Auslesen von Tags (Metadaten) von zahlreichen Dateiformaten in Kombination mit einer sehr hohen Verarbeitungsgeschwindigkeit gewährleistet werden. Ob Tags ausgelesen werden und wenn ja welche Daten ausgelesen werden sollen ist vollständig durch den Anwender konfigurierbar. Mittels einfacher Variablen lassen sich zudem auf schnelle Weise dynamische Titel generieren.

Der Playlist können Dateien jedes beliebigen Dateiformats hinzugefügt werden. Direkt nach der Installation ist bereits eine Vielzahl an verschiedenen Dateiformaten vordefiniert. Sollte das von Ihnen gewünschte Dateiformat fehlen, kann dieses im Handumdrehen hinzugefügt werden.

Für die Kompatibilität mit verschiedenster Hard- und Software kann Playlist Creator sowohl Playlisten im PLS-Format, als auch im M3U-Format erstellen.
